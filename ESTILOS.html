CSS - FOLHAS DE ESTILOS:

O CSS permite aplicar estilos seletivamente em documentos HTML. Por exemplo: para selecionar todos os elementos de parágrafos de uma página HTML e tornar o texto dentro deles vermelho: Toda essa estrutura é chamada de CONJUNTO DE REGRAS.

  • SELECTOR: Nome do elemento HTML <p>
  • DECLARATION: Uma regra como color: red; especificando quais das propriedades do elemento você quer estilizar.
  • PROPERTY: Forma pela qual você estiliza (no caso, color)
  • PROPERTY VALUE: O valor da propriedade. No caso, red.

    • IMPORTANTE:

  • 1. CADA LINHA DEVE SER ENVOLVIDA EM CHAVES <{}>
  • 2. DENTRO DE CADA DECLARAÇÃO, VOCÊ DEVE USAR DOIS PONTOS PARA SEPARAR A PROPRIEDADE DOS VALORES.
  • 3. DENTRO DE CADA CONJUNTO DE REGRAS, VOCÊ DEVE USAR UM PONTO E VÍRGULA PARA SEPARAR CADA DECLARAÇÃO DA PRÓXIMA.

    • MÚLTIPLOS ELEMENTOS:

    Você também pode selecionar vários tipos de elementos e aplicar um único conjunto de regras a todos eles. Inclua múltiplos seletores separados por vírgulas.

    DIFERENTES TIPOS DE SELETORES:

  • SELETOR DE ELEMENTO (tag): Todos os elementos HTML de determinado tipo.
    p seleciona <p>

  • SELETOR DE ID: O elemento na página com o ID especificado. É uma boa prática usar um elemento por ID (e claro, um ID por elemento) mesmo que seja permitido usar o mesmo ID para vários elementos.
    code>#id seleciona <p id="id">

  • SELETOR DE CLASSE: Os elementos na página com a classe especificada (várias instâncias de classe podem aparecer em uma página).
    .classe seleciona <p class="classe">

  • SELETOR DE ATRIBUTO: Os elementos na página com o atributo especificado.
    img[src] seleciona <img src="imagem.png"> mas não seleciona <img>

  • SELETOR DE PSEUDO-CLASSE:Os elementos especificados, mas somente quando estiver no estado especificado.
    Exemplo: com o mouse sobre ele.
    a:hover seleciona <a> somente quando estiver com o mouse sobre ele.

    • CAIXAS:

    Muitos dos elementos HTML podem ser pensados como caixas umas em cima das outras. O layout CSS é baseado principalmente no modelo de caixas. Cada um dos blocks que ocupam espaço na página tem propriedades como essas:

  • padding espaço ao redor do conteúdo.
  • border linha do lado de fora do padding.
  • margin espaço externo a um elemento.
  • width largura do elemento.
  • height altura do elemento.
  • background-color cor de fundo.
  • color cor do conteúdo (geralmente texto).
