ESTRUTURA DO HTML:

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.

As principais partes de um elemento são:

  • TAG DE ABERTURA: Consiste no nome do elemento envolvido em parênteses angulades de abertura e fechamento. Isso demostra onde o elemento começa ou onde seu efeito inicia.
  • TAG DE FECHAMENTO: Inclui uma barra antes do nome do elemento, demonstra onde ele acaba.
  • CONTEÚDO: O conteúdo do elemento.
  • ELEMENTO: A tag de abertura, fechamento e o conteúdo formam o elemento.

    • Elementos podem ter atributos, eles contém informações extra sobre o elemento que você não quer que apareça no conteúdo real. Por exemplo: <class="hidden"> class é o nome do atributo e hidden é o valor do atributo. Class permite que você forneça ao elemento um identificador que possa ser usado posteriormente para aplicar ao elemento informações de estilo e outras coisas.
    Um atributo sempre deve ter:

  • 1. Um espaço entre ele e o nome do elemento (ou atributo anterior, se o elemento já tiver um).
  • 2. O nome do atributo, seguido por um sinal de igual e aspas de abertura e fechamento, envolvendo o valor do atributo.

    • ANATOMIA DO HTML:

    <!DOCTYPE html> — Declara o tipo de documento como HTML5.
    <html></html> — Marca o início do documento HTML.
    <head></head> — Contém informações sobre o documento, como metadados e links para arquivos externos.
    <meta charset="utf-8"></meta> — Define o conjunto de caracteres usado no documento como UTF-8, que suporta uma ampla gama de caracteres.
    <title></title> — Define o título da página, exibido na barra de título do navegador.
    <body></body> — Contém o conteúdo visível da página.

    BODY: ESTRUTURAÇÃO:

    Cabeçalho: <header>.
    Barra de navegação: <nav>.
    Conteúdo principal: <main>, com várias subseções de conteúdo representadas por <article>, <section>, e elementos <div>.
    Rodapé: <footer>.

  • <main>:
    • é para o conteúdo único dessa página. Use <main> apenas uma vez por página, e o coloca diretamente dentro do <body>. Não é ideal aninhar ele dentro de qualquer outro elemento senão o elemento <body>.
  • <article>:
    • inclui um bloco de conteúdo relacionado o qual faz sentido por si só, sem o restante da página (por exemplo, uma postagem singular dum blog).
  • <section>:
    • é similar com <article>, mas ele é mais para agrupar uma única parte de página que constitui em um único pedaço de funcionalidade (por exemplo, um minimapa, ou um conjunto de manchetes e resumo). É considerado boa prática começar cada seção com um título; observe também que você pode dividir um <article>s em diferentes <section>s, ou <section>s em diferentes <article>s, dependendo do contexto.
  • <aside>:
    • é para conteúdo que não está relacionados diretamente com os conteúdos principais, mas que podem providenciar informações complementares a esses (entradas de glossários, biografia do autor, links relacionados, etc.).
  • <header>:
    • representa um grupo de conteúdo introdutório. Se ele for um elemento filho do <body>, Ele é um header global da página do site, mas se for um elemento filho de um <article> ou <section>, é definido como um header específico para essa seção (tenta não confundir isso com títulos e cabeçalhos).
  • <nav>:
    • contém a funcionalidade principal de navegação da página. Links secundários, etc., não iria na navegação.
  • <footer>:
    • representa um grupo de conteúdo final da página.
