Alguns dos principais recursos da linguagem, para entender melhor como o JavaScript funciona.

VARIÁVEIS:

São conteiners que armazenam valores. Você começa declarando uma variável com a palavra-chave let, seguida pelo nome que você dá variável.
let myVariable;
Um ponto e vírgula no final de uma linha indica onde uma instrução termina. Só é necessário quando você precisa separar instruções em uma única linha. No entanto, algumas pessoas acreditam que é uma boa prática colocar ponto-e-vírgula no final de cada instrução.

Você pode nomear uma variável para praticamente qualquer coisa, mas há algumas restrições. REGRAS SOBRE NOMENCLATURA

O JavaScript diferencia maíuscula de minúscula. Isso significa que myVariable não é o mesmo que myvariable.

Depois de declarar uma variável, você pode atribuir um valor:
myVariable = "bob";
Além disso, você pode fazer as duas operações na mesma linha:
let myVariable = "bob";
Você recupera o valor chamando o nome da variável:
myVariable;
Depois de atribuir um valor, você pode alterá-lo posteriormente no código:
myVariable = "bob";
myVariable = "bill";

Observe que as variáveis podem conter valores que tem diferentes tipos de dados:

  • STRING: Essa é uma sequência de texto conhecida como string. Para significar que o valor for uma string, coloque-a entre aspas simples.
    let myVariable = 'bob';

    NUMBER: Um número. Números não têm aspas.
    let myVariable = 1;

    BOOLEAN: Um valor que é true ou false.
    let myVariable = true;

    ARRAY: Permite armazenar vários valores em uma única referência.
    let myVariable = [1, 'bob', 'bill', 10]; Refere-se a cada membro do array assim: myVariable[0], myVariable[1], etc.

    OBJECT: Tudo em JavaScript é um objeto e pode ser armazenados em uma variável. Tenha isso em mente!
    let myVariable = document.querySelector('h1'); todos os exemplos acima também.

    • OPERADORES:

    Um símbolo matemático que produz um resultado baseado em dois valores ou variáveis. Exemplo:

  • ADIÇÃO:
    • Adicione dois números ou combine duas strings. (símbolo +)
    1 + 1;
    'bob' + 'bill';

  • SUBTRAÇÃO, MULTIPLICAÇÃO E DIVISÃO:
    • Eles fazem o que você espera que façam em matemática básica. (símbolos -, *, /)
    1 - 1;
    1 * 1;
    1 / 1;

  • ATRIBUIÇÃO:
    • Atribui valor a uma variável. (símbolo =)
    let myVariable = 1;

  • IGUALDADE DE ESCRITA:
    • Executa um teste para ver se dois valores são iguais. Retorna um resultado true ou false (booleano). (símbolo ===)
    let myVariable = 1;
    myVariable === 1;

  • NÃO, NÃO É IGUAL:
    • Retorna o valor logicamente oposto do que precede. Acontece um true em um false, etc. Quando é usado ao lado do operador de igualdade, o operador de negação testa se dois valores não são iguais. (símbolo !, !==)
    Para "not" a expressão básica é "true", mas o comparação retorna "false" porque nós a negamos:
    let myVariable = 1;
    !(myVariable === 1);
    "Não é igual" dá basicamente o mesmo resultado com diferentes sintaxes. Aqui estamos testando: "é myVariable NÃO igual a 3". Isso retorna false porque myVariable É igual a 3:
    let myVariable = 3;
    myVariable !== 3;

    EXPRESSÕES E OPERADORES

    CONDICIONAIS:

    Estruturas usadas para testar se uma expressão retorna verdadeira ou não. Uma forma comum de condicional é a instrução: if...else. Exemplo: A expressão dentro do if () é o teste. Usa o operador de igualdade de escrita para comparar a variável iceCream com a string chocolate para ver se as duas são iguais. Se retornar true, o primeiro bloco será executado. Se for false, o segundo bloco (else) será executado.

    FUNÇÕES:

    Functions são uma forma de empacotar a funcionalidade que você deseja reutilizar.

    EVENTOS:

    Essas são estruturas de código que detectam atividades no navegador e executam o código em resposta. O exemplo mais óbvio é lidar com o evento de click, que é acionado pelo navegador quando você clica em algo com o mouse. Existem várias maneiras de anexar um manipulador de eventos a um elemento. Aqui selecionamos <html> em seguida sua função addEventListener(), passando o nome do evento ('click') e uma função para executar quando acontecer.
